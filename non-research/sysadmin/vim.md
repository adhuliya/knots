Useful Ubuntu commands
=============================

### Spell checking

    # invoke a spell check
    :setlocal spell spelllang=us_en

    # toggle spell check
    :setlocal spell!

    # help
    :help spell-quickstart
    :help spell

    # useful commands when spell is active
    ]s -- next error
    [s -- prev error
    z= -- suggest correction
    zg -- add word to 'spellfile'
